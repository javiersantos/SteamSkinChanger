Steam Skin Changer for Linux
============================

Steam Skin Changer for Linux is a script for Linux systems to install third-party skins for Steam.

Developed by Javier Santos, @fjaviersantos

#### How to use Steam Skin Changer
```
$ sh metroforsteam.sh
```

#### Skins supported
* Metro for Steam
